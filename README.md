*Введение:*

Функциональность будет заключаться в программе `сборки и доставки заказов`. Назначение направлено на улучшение качества и удобства жизни `населения`.

*Примеры использования:*

- **Быстрый заказ** - при нажатии на кнопку поступит заказ сразу в даркстор
- **Отслеживание местоположения курьера** - при нажатии на кнопку появится местоположение курьера на карте
- **Чат с курьером** - при нажатии на кнопку вас перекинет в чат с курьером для уточнения деталей доставки
- **Оценка доставки** - при нажатии на кнопку вы сможете оценить качество доставки
- **Отмена заказа** - при нажатии на кнопку вы можете отменить заказ
- **Связь с поддержкой** - при нажатии на кнопку вы сможете связаться с поддержкой и решить вашу проблему
- **Жалоба** - при нажатии на кнопку вы можете оставить жалобу на курьера при необходимости

*Технические требования:*

- [ ] Операционная система: Android, IOS, Windows
- [ ] Браузер: Edge, Chrome,
- [ ] Android версии 12+б Ios версии 18+
